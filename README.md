# Gerador de Imagens com IA

Projeto desenvolvido para gerar imagens a partir de descrições textuais utilizando inteligência artificial.

---

## Sobre o Projeto

Este sistema permite que o usuário digite uma descrição (prompt) e receba uma imagem gerada automaticamente.

O objetivo é demonstrar o uso de tecnologias web (HTML, CSS e JavaScript) na criação de uma aplicação interativa e moderna.

---

## Funcionalidades

-  Geração de imagens a partir de texto  
-  Interface simples e intuitiva  
-  Histórico de prompts utilizados  
-  Feedback de carregamento (loading)  
-  Download da imagem gerada  
-  Sugestões de prompts prontos  

---

## Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- GitHub Pages
